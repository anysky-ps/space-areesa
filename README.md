# SPACE AREESA (スペース・アリーサ)

2D縦スクロール型シューティングゲーム。
アリーサを操作して、宇宙の各地に捕らわれた7人の仲間たちを救い出しましょう！

## 🚀 遊び方 / How to Play

友達に送られたURL（GitHub Pagesなど）を開くだけで、ブラウザですぐに遊べます。

### 操作方法
- **PCの場合**: 
  - マウスのドラッグ または カーソルキーで移動
  - クリック/スペースキーでショット（自動連射あり）
- **スマホ・タブレットの場合**:
  - 画面をスライドして移動
- **ポーズ (一時停止)**:
  - 画面右上の「||」ボタンを押すと中断・再開できます。

## 🌟 特徴
- **全ステージ構成**: 様々な敵や障害物を乗り越え、最後に待ち受けるボスを撃破せよ。
- **救出システム**: ステージをクリアするごとに仲間が一人ずつ解放されます。
- **シネマティック・エンディング**: 全ステージクリア後には、特別なスライドショーとメッセージが待っています。

## 🛠 開発環境 / Tech Stack
- **Language**: HTML5, CSS3, JavaScript (Vanilla JS)
- **Audio**: Web Audio API / MP3 Support
- **Graphics**: 2D Canvas API
